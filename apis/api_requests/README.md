# API Requests

Ejercicios y ejemplos de consumo de APIs usando Python y la librería equests.

## Contenido

- Ejemplo de petición GET a una API pública
- Ejemplo de petición POST enviando datos en formato JSON
- Manejo de respuestas y errores

## Cómo ejecutar

1. Instala las dependencias:
   `ash
   pip install requests
   ` 
2. Ejecuta los scripts de ejemplo:
   `ash
   python ejemplo_get.py
   python ejemplo_post.py
   ` 

## Aprendizajes

- Uso básico de la librería equests`n- Procesamiento de respuestas JSON
- Manejo de errores en solicitudes HTTP
