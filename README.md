# Python Learning Projects

Colección de ejercicios y proyectos de Python organizados por categorías.

## Categorías

- **APIs**: Ejercicios y proyectos relacionados con el consumo e implementación de APIs
- **Data Handling**: Proyectos enfocados en manipulación y procesamiento de datos
- **Basic Applications**: Aplicaciones sencillas con funcionalidades específicas
- **Web Projects**: Proyectos orientados a desarrollo web con Python

Cada proyecto contiene su propio README con instrucciones y explicaciones.
